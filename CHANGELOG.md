## [1.0.0] - 2025-12-26

### Added
- Initial release of GlassToolbar
- Liquid Glass style toolbar with blur effect
- Side button support with multiple preset styles
- Accessory view system with lifecycle management
- Space-adaptive layout with 5 space tiers
- Item priority system (essential/primary/secondary/overflow)
- Ultra minimal mode for space-constrained scenarios
- Nested corner radius auto-calculation
- Comprehensive appearance configuration via `ToolbarAppearanceConfiguration`
- Built-in accessory components (Demo):
  - `DualSliderAccessoryView`
  - `MiniPlayerAccessoryView`
  - `StatusAccessoryView`
  - `ProgressAccessoryView`
  - `QuickActionAccessoryView`
  - `FavoritesListAccessoryView`
  - `BrushListAccessoryView`
